📜 Python – Mémo de Référence pour les Exercices

📘 À garder à côté de soi pendant les exercices !

📨 print() – Afficher un message

print("Bonjour !")

➞ Affiche :
Bonjour !

💡 Pourquoi ?
Tu veux que ton programme parle ? C’est avec print() que tu lui fais afficher du texte ou des résultats.

📆 Variables – Mémoriser des infos

nom = "Alex"
age = 15
print(nom)

➞ Affiche :
Alex

💡 Pourquoi ?
Une variable est comme une boîte avec une étiquette : tu y ranges quelque chose et tu peux le réutiliser.

🎤 input() – Demander quelque chose à l'utilisateur

prenom = input("Quel est ton prénom ? ")
print("Bonjour", prenom)

➞ Si tu tapes Emma, il affiche :
Bonjour Emma

⚠️ Attention : input() renvoie toujours du texte. Si tu veux un nombre :

age = int(input("Quel âge as-tu ? "))

🧰 Fonctions – Blocs de code réutilisables

def dire_bonjour():
    print("Salut tout le monde !")

dire_bonjour()

➞ Affiche :
Salut tout le monde !

Avec paramètre :

def bonjour(prenom):
    print("Salut", prenom)

bonjour("Léo")

➞ Affiche :
Salut Léo

Avec retour :

def addition(a, b):
    return a + b

print(addition(3, 5))

➞ Affiche :
8

💡 Pourquoi ?
Une fonction, c’est une recette : tu l’écris une fois, tu peux la réutiliser partout.

🔎 Conditions – Faire des choix

age = 17

if age < 12:
    print("Tu es un enfant")
elif age < 18:
    print("Tu es un ado")
else:
    print("Tu es majeur")

➞ Affiche :
Tu es un ado

🔁 Boucle for – Répéter un certain nombre de fois

for i in range(3):
    print("Tour", i)

➞ Affiche :
Tour 0
Tour 1
Tour 2

prenom = "Emma"
for i in range(5):
    print("Bonjour", prenom)

➞ Affiche :
Bonjour Emma (5 fois)

🔄 Boucle while – Répéter tant qu'une condition est vraie

x = 3
while x > 0:
    print(x)
    x -= 1
print("BOOM")

➞ Affiche :
3
2
1
BOOM
